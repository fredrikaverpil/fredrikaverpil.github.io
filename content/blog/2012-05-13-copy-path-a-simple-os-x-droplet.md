Drop any file(s) or folder(s) onto this app and its path(s) will be put in the clipboard, ready for pasting (cmd+v). Place the app in the dock for easy access.
The source code was grabbed from Cole's [Mac OS X Hints post](https://hints.macworld.com/article.php?story=20071018002311820) and modified, so all credit goes to him!
